# 수신기 하드웨어

## 역할

무선 패킷을 수신해 순서와 손실을 처리하고 안전한 오디오 신호로 출력합니다.

## 필요 기능

- 무선 수신과 재연결
- 패킷 버전과 길이 검증
- 손실, 중복과 순서 역전 감지
- 지터 버퍼와 언더런 처리
- 디코딩 또는 PCM 재생
- I2S DAC 또는 아날로그 출력
- 볼륨 제한과 부팅 시 음소거
- 상태 표시
- 배터리, 충전과 저전압 처리
- 착용 가능한 크기와 무게

## 출력 후보

### 일반 유선 이어폰
- 장점: 비교와 교체가 쉬움
- 주의: 안전한 최대 출력과 이어폰 구동 회로 필요

### 골전도 출력
- 장점: 주변 소리 인지와 스포츠 용도에 적합할 가능성
- 주의: 구동 전력, 진동 전달 구조, 음질과 착용 시험 필요

### 기성 Bluetooth 이어폰 연계
- 장점: 착용 장치 개발 부담 감소
- 주의: 추가 무선 구간으로 지연과 복잡성이 증가할 수 있음

## 착용 원칙

- 배터리와 회로가 피부에 직접 닿지 않도록 절연합니다.
- 케이블 걸림과 흔들림을 보행 시험부터 확인합니다.
- 방수 성능은 실제 검증 전 주장하지 않습니다.
- 주변음을 들을 필요가 있는 장소에서는 음량을 제한합니다.

## 완료 기준

- M0 수신 통계와 버퍼 상태 제공
- 잘못된 패킷으로 큰 소리나 재부팅이 발생하지 않음
- 재부팅 후 송신 스트림 재탐색
- 30분 연속 출력과 온도 기록
