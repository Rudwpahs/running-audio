# 왜 이 리드부터인가

서울시는 2026-07-27~08-07에 **시립청소년시설 24개소, 약 350개 프로그램, 청소년 420명** 규모의 여름 동행캠프를 운영한다. 환경생태 분야에는 광진·강동·동대문·목동·망우·문래 등이 포함되고, 현장 체험·탐방·플로깅 같은 이동형 활동이 실제 프로그램에 들어간다.

Source:
- https://www.seoul.go.kr/news/news_report.do?bbsNo=158&nttNo=460647
- https://mediahub.seoul.go.kr/archives/2018617

따라서 `야외 그룹에서 진행자 안내가 참가자에게 잘 전달되는가`를 묻기에 현재 시점에서 관련성이 높은 운영자 집단이다.

다만 공공 청소년시설은 **첫 매출 고객으로 확정한 것이 아니다.** 초기 인터뷰를 통해 문제를 배우고, 구매 속도를 검증하기 위해 민간 러닝 운영자도 함께 넣었다.

# 통화 결과 코드

각 연락 후 반드시 아래 중 하나로 기록한다.

- `NO_ANSWER` — 연결 안 됨
- `GATEKEEPER` — 대표번호에서 담당자까지 못 감
- `REFERRED` — 담당자 이름/번호/메일 확보
- `INTERVIEW_BOOKED` — 날짜/시간 확정
- `INTERVIEW_DONE` — 인터뷰 완료
- `PILOT_INTEREST` — 실증 의향 있음
- `PAID_PILOT` — 구체적 금액+날짜 동의
- `NOT_A_PROBLEM` — 문제 자체가 약함
- `CURRENT_SOLUTION_GOOD` — 현재 대안에 만족
