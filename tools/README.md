# 개발·측정 도구

펌웨어와 독립적으로 시험을 재현하기 위한 도구를 둡니다.

## 예정 도구

- 패킷 송신률과 손실 로그 파서
- CSV 시험 결과 요약기
- 원본과 출력 오디오 지연 측정기
- 시험 톤과 클릭 신호 생성기
- 직렬 로그 수집기
- 프로토콜 인코더와 디코더 단위 테스트

## 원칙

- Python 도구는 가상환경에서 실행합니다.
- 원본 음원, 영상과 대용량 캡처는 저장소에 올리지 않습니다.
- 예제 입력은 저작권 문제가 없는 생성 신호를 사용합니다.
- 출력 보고서에는 실행 명령과 펌웨어 커밋을 기록합니다.
