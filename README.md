# Running Audio

스마트폰을 들고 뛰지 않아도 음악을 들을 수 있도록, 스마트폰의 오디오를 별도 송신기가 받아 착용형 수신기로 전달하는 저비용 무선 오디오 MVP 프로젝트입니다.

> 현재는 설계·기술 검증 단계입니다. 안전성과 품질이 검증된 제품이 아니므로 운동 중 착용이나 배터리 사용 전 충분한 시험이 필요합니다.

## 문제와 목표

달리기 중 스마트폰의 무게·흔들림·파손·분실 위험을 줄이기 위해 스마트폰은 안전한 위치에 두고 가벼운 수신기만 착용하는 구성을 검증합니다.

## 예상 흐름

1. 스마트폰을 송신기에 연결합니다.
2. 송신기가 오디오를 수집·패킷화합니다.
3. 외부 인터넷 없이 근거리 무선 링크로 수신기에 보냅니다.
4. 수신기는 오디오를 복원해 이어폰 또는 골전도 출력 장치로 전달합니다.

## MVP 범위

- 송신기 1대, 수신기 1대
- 집 안의 책상·방·벽 1개·복도 환경에서 검증
- 외부 Wi-Fi·인터넷에 의존하지 않는 구조 우선
- 고음질보다 연결 안정성·지연·제작비·재연결 능력 우선
- 향후 1:N 전송 가능성을 막지 않는 구조

## 확정 사항

- 송신기와 수신기를 별도로 제작합니다.
- 최초 검증은 집에서 진행합니다.
- 학생 예산을 고려한 저비용 모듈형 시제품부터 시작합니다.
- 모든 실험은 보드, 펌웨어 커밋, 거리, 장애물, 전원 조건과 함께 기록합니다.

## 미확정 사항

- 입력 방식: 3.5mm, USB-C, Bluetooth, 앱 기반 UDP
- 무선 방식: ESP-NOW, Wi-Fi SoftAP+UDP, Bluetooth, 전용 2.4GHz 모듈
- 보드, 코덱, 샘플레이트, 핀 배치, 회로, 배터리와 출력 방식

미확정 사항은 [`docs/decision-log.md`](docs/decision-log.md)에 근거와 재검토 조건을 기록합니다.

## 초기 검증 목표

아래 수치는 제품 사양이 아니라 첫 시제품의 검증 목표입니다.

- 실내 10m 이상 기본 연결 시험
- 반복적인 끊김 여부 측정
- 종단 간 지연 실측
- 최소 30분 연속 작동
- 재부팅 후 재연결
- 송신기 1대+수신기 1대 부품비 산출

## 안전

- 리튬 배터리는 보호회로·충전전류·발열·단락을 검토한 뒤 사용합니다.
- 주변음을 차단하면 위험할 수 있으므로 야외에서는 개방형·골전도 방식과 적절한 음량을 우선 검토합니다.
- 개인 음원, 영상, 비밀번호, 비밀키와 개인정보는 저장소에 올리지 않습니다.
